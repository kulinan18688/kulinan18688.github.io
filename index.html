不等式解答

要确定参数 ( a ) 的取值范围，使得不等式
[
x^2 + a x + 15 - a \geq 0
]
对于所有 ( x ) 满足 ( -4 \leq x \leq 0 ) 恒成立，我们可以按照以下步骤进行分析：

1. 分析二次函数的性质

不等式可以表示为二次函数：
[
f(x) = x^2 + a x + (15 - a)
]

因为二次项系数为正（即 ( 1 > 0 )），所以函数图像开口向上。为了保证不等式在区间 ( [-4, 0] ) 上恒成立，需要保证函数在该区间内的最小值不小于零。

2. 确定最小值的位置

二次函数的最小值出现在顶点处，即：
[
x = -\frac{a}{2}
]

根据 ( x ) 的取值范围 ( [-4, 0] )，可以分为以下几种情况：

情况一：当 ( a \in [0, 8] ) 时，顶点 ( x = -\frac{a}{2} ) 落在区间内

此时，函数在顶点处取得最小值：
[
f\left(-\frac{a}{2}\right) = \left(-\frac{a}{2}\right)^2 + a \left(-\frac{a}{2}\right) + 15 - a = -\frac{a^2}{4} + 15 - a
]
要求 ( f\left(-\frac{a}{2}\right) \geq 0 )：
[
-\frac{a^2}{4} + 15 - a \geq 0 \
\Rightarrow \frac{a^2}{4} + a - 15 \leq 0
]
解这个不等式：
[
a^2 + 4a - 60 \leq 0 \
\Rightarrow a \in [-10, 6]
]
结合 ( a \in [0, 8] )，得到 ( a \in [0, 6] )。

情况二：当 ( a < 0 ) 时，顶点 ( x = -\frac{a}{2} ) 不在区间内，最小值出现在 ( x = 0 )

此时：
[
f(0) = 15 - a \geq 0 \
\Rightarrow a \leq 15
]
由于 ( a < 0 )，因此 ( a \in (-\infty, 0) )。

情况三：当 ( a > 8 ) 时，顶点 ( x = -\frac{a}{2} ) 在区间外，最小值出现在 ( x = -4 )

此时：
[
f(-4) = (-4)^2 + a(-4) + 15 - a = 16 - 4a + 15 - a = 31 - 5a \geq 0 \
\Rightarrow a \leq \frac{31}{5} = 6.2
]
但是 ( a > 8 ) 与 ( a \leq 6.2 ) 无解，因此此情况无效。

3. 综合分析

结合以上情况，满足条件的 ( a ) 的取值范围为：
[
a \leq 6
]

最终结论

参数 ( a ) 的取值范围是所有实数中小于或等于 6 的数，即：
[
a \leq 6
]
